## 2. 隐私政策 (Privacy Policy)

**生效日期：2026年3月8日**

### 引言
“同频”（以下简称“本应用”）由 本软件开发者（以下简称“我们”）开发并运营。我们深知位置信息及应用使用数据的敏感性，本政策旨在说明我们如何收集、存储及保护您的信息。

### 我们如何收集和使用信息
为了实现本应用的核心功能（伴侣定位、足迹记录与应用锁定），我们需要收集以下信息：
* **位置信息**：为了生成“30天历史足迹”并实现实时位置共享，我们需要在后台持续收集您的经纬度坐标。该数据经过加密处理，且仅对您授权配对的伴侣可见。
* **账户与配对信息**：当您注册并与伴侣配对时，我们会收集您的设备标识符、配对码及对方的关联标识。
* **屏幕使用时间 (Family Controls)**：本应用使用 Apple 的 `Family Controls` 接口。我们仅利用此权限执行您主动设置的“应用限制”功能，我们**不会**收集、上传或分析您的具体 App 使用列表或偏好。
* **推送信息**：我们会收集您的设备推送 Token，用于向您的伴侣发送位置提醒或系统通知。

### 数据安全与存储
* **加密传输**：您的所有数据均通过 HTTPS 协议加密传输。
* **数据库安全**：位置与足迹数据存储于我们的生产数据库（MongoDB），并采取严格的访问控制。
* **运营主体**：本应用由个人开发者进行技术维护，严格遵守相关法律法规，保护用户隐私。

### 权限使用说明
* **始终位置权限**：用于在后台持续记录并分享足迹。
* **网络权限**：用于将坐标安全同步至云端，并获取伴侣的最新位置。
* **屏幕使用时间权限**：用于实现应用锁定逻辑，防止在设定时间内过度使用特定 App。

### 订阅服务说明 (In-App Purchase)
* **服务内容**：本应用提供 Pro 会员订阅，解锁 30 天足迹查询等高级功能。
* **自动续费**：订阅将通过您的 iTunes 账户扣费。您可以随时在 iOS 系统设置中的“Apple ID”->“订阅”里管理或取消自动续费。

### 信息共享与披露
我们承诺不会将您的个人位置数据或应用锁定设置出售或共享给任何第三方。除非涉及以下特殊情况：
1. 获得您的明确授权；
2. 法律法规、法律程序或政府强制性要求。
